「按时吃药」（以下简称"本应用"）由个人开发者开发和运营，包名为 uni.app.UNI25F811F。本隐私政策适用于您使用本应用的全部场景。

在您使用本应用前，请认真阅读本隐私政策。您一旦开始使用本应用，即表示您已同意本政策的全部内容。如果您不同意本政策，请停止使用本应用。

我们承诺：严格遵守《中华人民共和国个人信息保护法》《网络安全法》以及华为应用市场相关规范，最小化收集您的个人信息，保障您的合法权益。

📊二、我们收集的信息

2.1 我们收集的信息类型

信息类型 具体内容 收集目的 是否必须
设备标识 推送客户端标识（CID），由个推 SDK 生成，不含真实设备 ID 发送药物提醒推送通知 是
提醒设置 您设定的服药提醒时间（如 08:00、20:00） 在指定时间发送提醒 是
服药记录 您标记的当日是否已服药（是/否） 避免重复提醒，提供服药统计 否（可选功能）
语言偏好 中文/英文界面选择 推送通知使用对应语言 否

2.2 我们不收集的信息

  • 不收集您的姓名、手机号、身份证号等个人身份信息
  • 不收集您的精确地理位置信息
  • 不收集您的通讯录、相册、摄像头等敏感权限数据
  • 不收集您的药物名称、病历、健康诊断等医疗信息
  • 不追踪您在其他 App 中的行为

2.3 信息存储位置

您的提醒时间和服药记录仅存储在您的手机本地，不上传至服务器。推送标识（CID）存储在 uniCloud 云数据库中，用于向您的设备发送提醒通知，服务器位于中国境内（阿里云）。

🎯三、信息的使用方式

我们仅将收集到的信息用于以下目的：

  1. 发送药物提醒通知：根据您设定的时间，通过推送服务向您的设备发送服药提醒。
  2. 避免重复提醒：记录当日是否已提醒，防止同一时间多次推送。
  3. 功能改善：分析推送到达率等匿名统计数据，用于改善提醒的可靠性。
⚠️ 我们不会将您的信息用于广告推送、用户画像分析、出售给第三方，或任何与药物提醒无关的用途。

🔐五、应用权限说明

本应用仅申请以下系统权限，不申请任何不必要的敏感权限：

权限名称 用途说明 是否必须
通知权限（POST_NOTIFICATIONS） 向您发送服药提醒通知 是（核心功能）
精确闹钟（SCHEDULE_EXACT_ALARM） 在精确时间触发服药提醒 是（核心功能）
开机自启（RECEIVE_BOOT_COMPLETED） 手机重启后自动恢复提醒计划 是（保证提醒连续性）
前台服务（FOREGROUND_SERVICE） 保持提醒服务在后台运行，防止被系统终止 是
忽略电池优化（REQUEST_IGNORE_BATTERY_OPTIMIZATIONS） 防止系统省电策略阻止提醒推送 建议开启（用户可拒绝）
网络访问（INTERNET） 同步提醒设置、接收推送通知 是

本应用不申请位置、摄像头、通讯录、麦克风、存储等敏感权限。

🗑️六、数据保留与删除

6.1 保留期限

  • 本地数据（提醒时间、服药记录）：存储在您的设备上，卸载应用后自动清除。

  • 卸载本应用，本地数据立即清除
  • 通过下方联系方式发送数据删除请求，我们将在 15个工作日内处理
